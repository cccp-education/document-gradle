== Décisions techniques

=== Refactor GitReleaseNotesGenerator — configDriven factory

Le contrat N0 `ReleaseNotesGenerator.renderer` est non-null. Pour supporter
la sélection par `rendererType` sans casser le contrat, le constructeur
principal est privé, deux constructeurs publics assurent la compat
ascendante (DOC-8.1), et une factory `configDriven()` active le mode
sélection-dynamique. L'extension du fichier de sortie est déduite de
`renderer.format` (`md`/`json`/`adoc`).

=== Categories custom — fallback config default

`ReleaseNotesDsl.categories` est une `MapProperty<String,String>` vide par
défaut. La task passe `categories.get().takeIf { it.isNotEmpty() } ?:
ReleaseNotesConfig().categories` — fallback sur les 7 catégories Conventional
Commit par défaut du contrat N0. Évite de dupliquer les défauts dans le DSL.

=== CLI override

`-Pdocument.releaseNotesRendererType=markdown` et
`-Pdocument.releaseNotesCategories=feat=New,fix=Bugs` surchargent le DSL.
Pattern de précédence respecté : CLI > DSL > convention.

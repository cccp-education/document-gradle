== Decisions architecturales

. *PlantUML = composition, pas parallelisme* ‚Äî Un diagramme est un constituant atomique du document (comme `<img>` en HTML). Dependance compileOnly legitime, pas de cycle DAG.
. *Codex fait aussi du LLM* ‚Äî OCR photos ‚Üí AsciiDoc structure via deepseek-v4-pro. Contrats `codex.ocr` deja implementes (session 027).
. *Document = 3 missions* ‚Äî Creation (LLM), enrichissement (PlantUML/images/HTML/includes), publication (HTML/PDF/EPUB/DocBook/ManPage).
. *Theme = fichiers versionnes* ‚Äî Document applique le theme (YML PDF, CSS HTML/EPUB), ne le cree pas.
. *Pont N2‚ÜîN2 = AsciiDoc sur disque* ‚Äî Codex ecrit, Document lit. Aucune dependance Gradle directe.

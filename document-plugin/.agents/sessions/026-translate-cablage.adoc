== Usage

[source,shell]
----
./gradlew translateDocument -Pdocument.translateSource=article.adoc -Pdocument.translateSourceLang=fr -Pdocument.translateTargetLang=en -Pdocument.translateLlmMode=fake
----

DSL :
[source,kotlin]
----
document {
    translation {
        sourceFile.set("src/docs/article.adoc")
        sourceLanguage.set("fr")
        targetLanguage.set("en")
        llmMode.set("ollama")
    }
}
----

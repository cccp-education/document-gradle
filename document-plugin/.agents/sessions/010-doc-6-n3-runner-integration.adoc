== Contrat N3 respecté

Le contrat N3 de facto (pas de contrat N0 formel — chaque borough duplique
une classe `Metadata`) :

.`composite-context.json` (shape)
[source,json]
----
{
  "source" : "new-orleans",
  "entries" : [
    {
      "source" : "document",
      "path" : "/abs/path/document.html",
      "format" : "html",
      "backend" : "html5",
      "sourceAdoc" : "source.adoc",
      "exists" : true
    }
  ],
  "count" : 1,
  "timestamp" : 1751558400000
}
----

.`metadata.json` (shape)
[source,json]
----
{
  "source" : "new-orleans",
  "type" : "retrieve",
  "sessions" : 1,
  "generatedAt" : "2026-07-03T13:51:47.123Z",
  "model" : "onnx-local",
  "version" : "1.0",
  "dependencies" : [ "brooklyn", "htown" ]
}
----

Runner-gradle lit uniquement la clé `entries` du `composite-context.json`.
Le `metadata.json` est la provenance (audit trail).

== Loi de l'Économie d'Encre

`collectDocumentRetrieve` est READ-only : il indexe les artefacts produits
par les autres tâches (generateDocument, enrichDocument, convertXxx).
Il n'invoque jamais le LLM et ne régénère jamais un document — il snapshot
ce qui existe sur disque.

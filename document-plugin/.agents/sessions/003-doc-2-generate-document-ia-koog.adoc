==== GenerateDocumentTask (DOC-2 réel)

Deux modes :
1. **Mode IA** (si `prompt` set) : graphe koog → LLM → AsciiDoc validé → fichier `.adoc`
   - Économie d'encre : hash SHA-256 du prompt stocké en metadata du fichier généré.
     Skip si sortie existante avec même hash (pas de re-invocation LLM).
2. **Mode copie** (sinon) : fallback DOC-1 (copie source vers sortie).

Extension `document { }` : +prompt, +llmMode ("ollama"|"fake"), +systemPrompt.
Convention `source` retirée (la source n'est requise qu'en mode copie, pas en mode IA).

== Décisions techniques

. **koog sur langchain4j** : aligné sur codebase-gradle et KOOG_AGENTIC_PATTERNS.adoc.
   koog orchestre (strategy, node, edge, onCondition), langchain4j exécute (OllamaChatModel).
. **FakeDocumentLlmProvider dans src/main** : le plugin expose `llmMode = "fake"` au build script,
   donc le fake doit être dans le classpath main (pas seulement test). Permet tests
   GradleTestKit/Cucumber sans Ollama ni clé API.
. **Convention source retirée** : en mode IA, la source n'est pas une input. `@InputFile @Optional`
   sans convention évite la validation Gradle d'un fichier inexistant.
. **Économie d'encre** : hash SHA-256 du prompt stocké en commentaire dans le fichier généré.
   Skip si hash identique → pas de re-invocation LLM metered.

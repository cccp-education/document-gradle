== Décisions techniques

- `parseLines` reverse l'ordre car `git log` retourne newest-first ; on veut chronologique (oldest-first)
- `detectFromTag` utilise `--sort=-v:refname` (version sort desc) car les lightweight tags n'ont pas de creatordate
- Extension `.adoc` (pas `.asciidoc`) — convention workspace
- `@DisableCachingByDefault` sur ReleaseNotesGenerateTask (git log non-déterministe)
- DSL nested `releaseNotes { }` pattern DOC-12 (classe concrète, val Property, fun action(Action<T>))
- CLI override : `-Pdocument.releaseNotesFromTag`, `-Pdocument.releaseNotesToTag`, `-Pdocument.releaseNotesVersion`, `-Pdocument.releaseNotesIncludeDownloads`

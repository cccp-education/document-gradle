== DSL unifié (session 013, stable)

```
document {
    source.set(file("livre.adoc"))
    enrich { plantuml.set(true); images.set(true); passthrough.set(true) }
    outputs { html.set(true); pdf.set(true); epub.set(true) }
    theme { pdfTheme.set(file("theme.yml")); htmlStylesheet.set(file("style.css")) }
    metadata { title.set("Mon Livre"); author.set("Auteur"); language.set("fr") }
}
```

- `serializeDocumentConfig` → `document-config.json` (contrat N3)
- Pattern nested : `fun enrich(action: Action<DocumentEnrichDsl>)` (org.gradle.api.Action)
- Classes DSL concrètes avec `val` initialisés via `project.objects.property()`
